# Handoff: Дизайн-система «Единый портал поддержки бизнеса»

> АО «НУХ «Байтерек» — единое цифровое окно к 70+ мерам господдержки от 7 дочерних организаций.

## Overview
Полная дизайн-система государственного/квазигосударственного веб-портала: токены (цвет, типографика, spacing, радиусы, тени), библиотека компонентов со всеми состояниями и 6 собранных экранов. Цель — строгий официальный госсервис (не стартап/SaaS), зелёно-золотая палитра монумента «Байтерек», без единого синего оттенка.

## About the Design Files
Файл `Дизайн-система.dc.html` — **дизайн-референс, созданный в HTML** (прототип, показывающий внешний вид и поведение), **а не продакшн-код для прямого копирования**. Задача — **воссоздать эти дизайны в целевом окружении** (рекомендуется Next.js + TypeScript + React, как указано заказчиком) на его паттернах и библиотеках. Вся визуальная система построена на **CSS Custom Properties** — их можно перенести почти дословно в `app/globals.css`.

> Технические детали: `.dc.html` использует внутренний рантайм (`support.js`) для стриминга и интерактивности — он **не нужен** в продакшене. Смотрите на разметку, инлайновые стили и токены как на спецификацию; интерактивную логику (переключатель шрифта, шаги формы) реализуйте средствами React.

## Fidelity
**High-fidelity (hifi).** Финальные цвета, типографика, spacing и состояния. Воссоздавать пиксель-в-пиксель, используя библиотеки и паттерны целевого кодовой базы.

## Components (со всеми состояниями)

### Кнопки
Базовые: `border:none; border-radius:var(--radius-sm); font:500 14px Golos Text; padding:11px 22px; cursor:pointer`. Мин. hit-area 44px для large.
- **Primary** — bg `--color-primary`, text #fff, `box-shadow:var(--shadow-sm)`. hover→`--color-primary-hover`; active→`--color-primary-active`; focus→`+var(--focus)`; disabled→bg `--green-100`, text `#8FB3A1`, `cursor:not-allowed`.
- **Secondary (gold)** — bg `--gold-500`, text **#000**. hover→`--gold-600`; active→`--gold-700` text #fff; disabled→`--gold-100` text `#BDAE7E`.
- **Outline** — transparent, `border:1.5px solid --color-primary`, text primary; hover bg `--green-50`.
- **Ghost** — transparent, text primary, без бордера; hover bg `--green-50`.
- **Danger** — bg `--error`, text #fff.
- Размеры: sm `7px 14px / 13px`, md `11px 22px / 14px`, lg `14px 30px / 16px`.

### Поля форм
`border:1.5px solid --border-strong; border-radius:var(--radius-sm); padding:11px 14px; font:15px`. Label: 13px/500 `--text-strong`, `margin-bottom:6px`. Helper: 12px `--text-muted`.
- **focus** — `border-color:--color-primary; box-shadow:var(--focus)`.
- **error** — `border-color:--error`, текст ошибки 12px `--error` с иконкой ✕ (не только цвет).
- **disabled** — bg `--bg-sunken`, text `--text-faint`, `cursor:not-allowed`.
- **select** — `appearance:none`, кастомная стрелка ▾ absolute right.
- **checkbox** 20px `--radius-xs`, checked bg `--color-primary` + ✓ #fff.
- **radio** 20px круг, checked — внутренняя точка 10px `--color-primary`.
- **toggle** — pill 38×22, on bg `--color-primary`, ручка 18px #fff.

### Статус-бейджи (pill)
`display:inline-flex; gap:7px; border-radius:var(--radius-pill); padding:6px 14px; font:500 13px`. Всегда **иконка (700) + текст**:
- Одобрено: bg `--success-bg`, text `--success`, border `#BEDBC9`, ✓
- На рассмотрении: bg `--warning-bg`, text `--warning`, border `#E6D9AC`, ● (`--warning-dot`)
- Требует внимания: те же + ⚠
- Отклонено: bg `--error-bg`, text `--error`, border `#EBC9C4`, ✕
- Черновик: bg `--info-bg`, text `--info`, ○

### Alert-баннеры
`border:1px solid <tint>; border-left:4px solid <semantic>; border-radius:var(--radius-sm); padding:14px 16px`. Иконка + заголовок 14px/600 + текст 13px.

### Breadcrumbs (обязательны по ТЗ)
`display:flex; gap:8px; font:14px`. Ссылки `--color-primary`, разделитель `›` цвета `--text-faint`, текущая страница `--text-muted` (не ссылка).

### Пагинация
Кнопки 38×38, `--radius-sm`. Активная — bg `--color-primary` text #fff/600. Неактивные — border `--border-strong`, hover bg `--green-50`. Стрелки ‹ ›, многоточие … цвета `--text-faint`.

### Header (шапка)
`padding:12px 20px; bg:#fff; border-bottom:1px solid --border`. Слева: логотип 44px + «Единый портал / поддержки бизнеса». Навигация 14px (active `--color-primary`/500). Справа: переключатель «А А», кнопки «?» и поиск (38×38, hover green-50), зелёная CTA «Личный кабинет», **золотая квадратная кнопка-бургер 40×40** (`--gold-500`, 3 полоски #000) — открывает мегаменю.

### Переключатель размера шрифта «А А» (WCAG 4.7)
Сегментированный контрол из 3 кнопок (А / А / А разного кегля) в бордере `--radius-sm`. Активный уровень — bg `--green-50`, text `--color-primary`. Управляет масштабом контента: уровни ≈ **15px / 17px / 20px** базового размера. Реализовать как множитель на корне контента.

## Interactions & Behavior
- **Переключатель «А А»**: 3 уровня масштаба базового шрифта (~15/17/20px), применяется к корню контента. Персистить выбор (localStorage).
- **Бургер → мегаменю**: полноэкранный оверлей, закрытие по × / Esc / клику вне.
- **Многошаговая форма**: state `currentStep` (1–4); «Далее» инкремент (clamp 4), «Назад» декремент (clamp 1); прогресс и контент реактивны; на шаге 4 CTA меняет текст.
- **Виджет-консультант**: toggle раскрытия карточки.
- **Карточки/кнопки/поля**: hover/active/focus по токенам выше. **focus-visible обязателен** (`var(--focus)`) — навигация с клавиатуры.
- **Таблица заявок**: статусы всегда цвет + иконка + текст.
- Responsive, **mobile-first**: ПК/планшет/мобильный. На мобильном — сайдбары сворачиваются, сетки карточек в 1 колонку, навигация header уходит в мегаменю.

- Переключатель размера шрифта в шапке (реализован).

- Статус — не только цветом (иконка + текст).
- Поддержка экранных дикторов (семантические теги, aria-label на иконочных кнопках), навигация без мыши.
